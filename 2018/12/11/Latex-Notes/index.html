Latex Notes

Some basic knowledge

Several consecutive whitespace characters are treated as one “space”.Whitespace at the start of a line is generally ignored, and a single line break is treated as “whitespace”.

Special Characters

Watch out! Backslash itself is different.

New case-sensitive command

will usually omit the whitespace after the command name. You can use {} to avoid that.

\command [Optional parameter] {parameter} \Question Optional parameter for?

Comment Or “Toggle”

% is different from the comment sign used in Java or C. It will ignore the rest of this line, the new line separator and all the whitespace of next line.

pagestyle

this command can set all the pages or current page have heading or footer

page breaker \newpage

\linebreak[n] \nolinebreak[n] can help us set the break line numbers

Reference and Tag

If you are using ‘bigger’ method, the equation will automatically have their order number.

But if you want to use other add a \tag{whateveryouwant} after the equation.

You can add a “label” after the equation and refer to it in the article.

Usage: \label{fuck} \eqref{fuck}

IF YOU DON’T WANT TO NUMBER THE EQUATION YOU CAN \begin{equation*}
